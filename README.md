# スグ履歴書 — 登録不要・個人情報を送信しない履歴書作成ツール

ブラウザだけで完結する、転職・就活向けの履歴書メーカー。
サーバー不要の静的サイトなので、**運用コスト0円**で公開できます。
「[スグ請求](../sugu-seikyu/)」に続く「スグ」シリーズ第2弾。

## 機能

- JIS様式風のA4・2ページレイアウト(印刷ダイアログからPDF保存)
- **和暦/西暦の切り替え**(令和・平成・昭和を改元月まで考慮して自動変換)
- **生年月日から入学・卒業年を自動計算**(早生まれ対応。「履歴書 卒業年 計算」は定番の検索ニーズ)
- 記入日時点の**満年齢を自動計算**
- **証明写真の挿入**: 端末内で縦4:横3に自動トリミング。画像はアップロードされない
- 学歴・職歴の「学歴」「職歴」「現在に至る」「以上」を自動配置
- 入力内容・写真は `localStorage` に自動保存(サーバー送信なし)

## なぜ履歴書か

- 履歴書は氏名・住所・生年月日・学歴・顔写真のかたまり = **「サーバーに送信されません」の訴求が請求書以上に刺さる**。大手サービスはほぼ全て会員登録が必要
- 「履歴書 テンプレート 無料」「履歴書 作成 無料 登録不要」は検索ボリュームが大きく、確定申告のような季節性がない(3〜4月・9〜10月に山はある)
- 利用者=転職見込み客なので、下記の高単価アフィリエイトに直結する

## 使い方

ビルド不要。`index.html` をブラウザで開くだけで動きます。

```sh
npx serve .
```

## デプロイ(無料)

- **GitHub Pages / Vercel / Netlify / Cloudflare Pages**: フォルダをそのままインポート(ビルド設定不要)
- スグ請求と同一ドメインのサブディレクトリ(例: `example.com/rirekisho/`)に置くと、シリーズ間の相互リンクでドメイン評価を集約できる

## 収益化プラン

優先度順。**1 → 2 が先**(実装が数時間で済み、審査が通りやすい)。

### 1. 転職エージェント・転職サイトのアフィリエイト(最有力)

このツールの利用者は「今まさに応募書類を作っている求職者」= 人材各社が最も欲しい見込み客。
**会計ソフトより単価が高く、1件 3,000円〜15,000円超**の案件が多い。

- リクルートエージェント / doda / マイナビ転職 / ビズリーチ / リクナビNEXT など(A8.net・バリューコマース・もしも経由)
- 設置場所: フッター、「印刷 / PDF保存」ボタン押下後(書類が完成した瞬間が最も転職意欲が高い)、SEO記事内
- 文言例: 「履歴書ができたら、次は求人探し。○○なら履歴書がそのまま使い回せます」

### 2. Google AdSense

- フォーム下部とフッターに広告枠(印刷プレビュー・印刷出力には出さないこと)
- 履歴書は滞在時間が長い(書き上げるのに20〜40分)ため、1訪問あたりの表示回数を稼ぎやすい

### 3. プレミアム機能(買い切り or 月額、Stripe Payment Links)

無料版はこのまま、以下を有料化:

- **職務経歴書メーカー**(転職者は必ずセットで必要になる。最優先の拡張)
- 複数の履歴書を保存・切り替え(応募先ごとの出し分け)
- デザインテンプレート切替(新卒向け・転職向け・アルバイト向け)
- 送付状(添え状)の自動生成

Stripe Payment Links ならバックエンド不要で決済ページを発行できます。

### 4. SEOコンテンツ

`/columns/` 以下に静的HTMLで記事を追加し、検索流入 → ツール利用 → アフィリエイト成約の導線を作る:

- 「履歴書の学歴はいつから書く?」「卒業年度 早見表」(自動計算機能への導線)
- 「本人希望記入欄には何を書く?」「志望動機の例文」
- 「履歴書とWebエントリーの違い」など

### 5. シリーズ相互送客

スグ請求(フリーランス向け)⇔ スグ履歴書(転職者向け)をフッターで相互リンク。
「フリーランス→就職」「会社員→独立」の両方向の動線が作れる。

## 技術構成

- Vanilla HTML / CSS / JavaScript(依存ゼロ、ビルドなし)
- `index.html` — フォームと履歴書プレビュー(A4×2枚)
- `css/style.css` — 画面用 + `@media print` の印刷用スタイル
- `js/app.js` — 和暦変換・年齢/卒業年計算・写真トリミング・localStorage 保存
